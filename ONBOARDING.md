# Onboarding

> Placeholder — onboarding guide for new team members.

## Welcome

This document walks you through setting up and using the Nanovest Agentic System.

## Prerequisites

- [ ] Access to the repository
- [ ] Environment variables configured (see `config/.env.example`)
- [ ] Claude Code installed

## Setup Steps

1. Clone the repository
2. Copy `config/.env.example` to `config/.env` and fill in values
3. Review `CLAUDE.md` for project-specific instructions
4. Read `AGENTS.md` to understand available agents
5. Review your team's `CLAUDE.md` under `teams/`
